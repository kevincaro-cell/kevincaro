Descripción

Este programa está hecho en Python y permite traducir texto en español a Braille.
El usuario ingresa una palabra, frase o número y el programa muestra la traducción en Braille.

Además, el resultado se guarda en un archivo de texto.

Cómo funciona

El usuario escribe un texto.

El programa convierte cada letra o número a su equivalente en Braille.

La traducción se muestra en pantalla.

También se guarda en el archivo braille.txt.
